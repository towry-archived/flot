Flot theme
----------

Flot is a theme for Wordpress. The design is inspired by someone's tumblr site that I have forgot.

And it's based on _s.

## Installation

```bash
$ git clone https://github.com/towry/flot flot
$ mv flot -t path/../wp-content/themes/
```
